# Student Management System

A Java console-based project to manage student records.

## Features
- Add student
- View students
- Search student
- Menu driven system

## Tech Stack
- Java
- OOP
- ArrayList
- Scanner

## How to Run
1. Open project in IntelliJ
2. Run Main.java
