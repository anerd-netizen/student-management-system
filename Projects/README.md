# 🎓 Student Management System

A simple Java-based console application to manage student records.
Built as part of a structured **DSA + Project learning journey**.

---

## 🚀 Features

* ➕ Add Student
* 📋 View Students
* 🔍 Search Student by ID
* ❌ Delete Student
* ✏️ Update Student
* 📊 Sort Students by Marks
* 📈 Count Students Above Threshold

---

## 🛠 Tech Stack

* Java
* ArrayList
* Basic DSA (Searching, Sorting, Frequency concepts)

---

## 🧠 Concepts Used

* Arrays & ArrayList
* Binary Search (logic understanding)
* Bubble Sort (with optimization)
* Prefix Sum (for problem-solving)
* Frequency Counting
* Clean Code Structure (methods separation)
